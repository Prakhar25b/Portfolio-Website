# Prakhar Bhatnagar — Personal Portfolio Website

A modern, animated, single-page personal portfolio website built with pure HTML, CSS, and JavaScript. Designed with a dark navy theme and smooth UI interactions to showcase skills, education, and projects professionally.

👉 To view : https://prakhar25b.github.io/Portfolio-Website/

---

## 🌐 Live Sections

- **Home** — Hero section with animated intro and social links
- **About** — Personal background and key info
- **Skills** — Tech stack displayed as interactive cards
- **Education** — Timeline of academic qualifications
- **Projects** — Featured project cards with tech tags
- **Interests** — Hobbies and extracurricular activities
- **Contact** — Phone, email, and location details

---

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Flexbox, Grid, keyframe animations
- **JavaScript (Vanilla)** — Scroll effects, IntersectionObserver, mobile nav, particles
- **Font Awesome 6.4** — Icons throughout the UI

---

## ✨ Features

- Fully responsive — mobile, tablet, and desktop
- Sticky navbar with active link highlighting on scroll
- Animated hero section with floating particles
- Scroll-triggered fade-in animations via IntersectionObserver
- Smooth scrolling navigation
- Hamburger menu for mobile
- Scroll-to-top button
- Hover micro-interactions on cards and icons

---

portfolio/

## 🚀 Getting Started

No build tools or dependencies required. Just clone and open in a browser:
```bash
git clone https://github.com/Prakhar25b/portfolio.git
cd portfolio
open index.html
```
